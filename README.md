# Qodex-SQL-ERD

Qodex-SQL-ERD is a comprehensive Entity-Relationship Diagram (ERD) tool designed to help database designers and developers visualize and manage their database schemas effectively. This tool provides an intuitive interface for creating, editing, and exporting ER diagrams, making it easier to understand complex database structures.

## structures

The ERD tool supports various database structures, including:

- Tables
- Relationships (One-to-One, One-to-Many, Many-to-Many)
- Attributes (Primary Keys, Foreign Keys, Unique Constraints)
